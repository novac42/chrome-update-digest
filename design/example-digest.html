Chrome 138 企业版摘要

最新功能亮点与企业安全更新

核心亮点

生产力亮点

  • 协作标签组 (Shared tab group)：桌面端和移动端支持协作标签组，允许多用户跨设备共享和同步标签页。
  • 智能搜索增强：地址栏基于页面内容的上下文搜索 + Agentspace 企业内部搜索建议
  • AI 能力集成：文本摘要、语言检测、翻译 API 和 AI 搜索推荐
  • 历史同步优化：Profile pill 入口提升历史同步可见性（企业用户 4 小时后可见）

移动企业安全亮点

  • iOS URL 过滤：支持网址/类别的审计、警告和阻止功能
  • Android APK 安全检查 Chrome 139：恶意 APK 自动检测和阻止
  • 增强型安全浏览同步：跨设备自动同步安全浏览设置
  • 配置文件自动清理：不活跃托管配置文件自动删除（默认 90 天）

即将到来的变更

  • Chrome 139：Android 恶意 APK 下载检查、新标签页管理披露信息显示、Gemini in Chrome 稳定版推广
  • Chrome 140：所有受管浏览器默认显示管理信息"Managed by [domain name]"
  • Android 系统支持变更：Chrome 139 起不再支持 Android Oreo/Pie，企业需规划系统升级

桌面端用户生产力更新

Chrome 138 当前稳定版本

使用 Profile Pill 选择历史同步

Chrome 138 桌面端部分已登录用户可通过 profile pill 入口选择开启历史和标签同步。企业用户在 4 小时无操作后才会看到该入口。此举旨在为用户提供更直观、非侵入式的历史同步体验，相关企业策略（如 SyncDisabled、SyncTypesListDisabled、SavingBrowserHistoryDisabled）依然有效。

协作标签组

Chrome 138 桌面端和移动端支持协作标签组，允许多用户跨设备共享和同步标签页。企业可通过 TabGroupSharingSettings 策略管理。

Chrome 地址栏上下文搜索建议

Chrome 138 桌面端支持在地址栏直接基于当前页面内容进行搜索，结合 Google Lens，可通过选中页面内容或输入问题获得上下文建议。该功能受 LensOverlaySettings 策略控制。

Chrome 搜索栏中的 Agentspace 推荐

Chrome 138 桌面端企业用户可在地址栏和新标签页搜索框中获得 Agentspace 内部搜索建议（如人员、文件、查询等），支持自定义关键字触发。

AI 能力 API

Chrome 138 桌面端引入多项 AI 能力，包括文本摘要、语言检测和翻译，均可通过企业策略 GenAILocalFoundationalModelSettings 控制。

即将到来的变更

Chrome 139：Gemini in Chrome 功能逐步推广至美国稳定版用户，支持文本和语音交互，可通过 GeminiSettings 或 GenAiDefaultSettings 策略控制

Chrome 移动端安全更新 (Android/iOS)

Chrome 138 移动端安全

iOS URL 过滤功能

Chrome 138 iOS 端支持 URL 过滤，企业可对特定网址或类别进行审计、警告或阻止，提升移动端安全性。

增强型安全浏览同步设置

Chrome 138 移动端（Android/iOS）增强型安全浏览现已同步设置，用户在任一设备开启后，所有已登录设备均自动应用。

不活跃配置文件自动删除

Chrome 138 移动端（Android）支持自动删除长期不活跃的托管配置文件，默认 90 天，管理员可自定义。

即将到来的变更

  • Chrome 139：Android 恶意 APK 下载检查功能（可通过 SafeBrowsingProtectionLevel 策略关闭）
  • Chrome 139：不再支持 Android Oreo/Pie，建议企业及时升级终端系统
