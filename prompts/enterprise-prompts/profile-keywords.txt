# Chrome Profile相关功能关键词组合
# 用于搜索Chrome更新文档中与profile相关的内容
# 已去除容易误召回的通用词汇如deletion、inactive、managed、browser、user等

# 核心关键词组合（正则表达式格式）
profile|Profile|account|Account|identity|Identity|sync|Sync|sign-in|signin|multiple.*account|separate.*profile|data.*separation

# 分解说明：
# profile|Profile - 直接匹配profile相关内容
# account|Account - 账户管理功能
# identity|Identity - 身份管理和认证
# sync|Sync - 同步功能
# sign-in|signin - 登录相关功能
# multiple.*account - 多账户支持
# separate.*profile - 独立配置文件
# data.*separation - 数据分离功能

# 注意事项：
# - 已排除deletion和inactive（容易匹配到非profile相关的删除和非活跃功能）
# - 已排除managed（太宽泛，会匹配到所有企业管理功能）
# - 已排除browser和user（过于通用的词汇）
# - 保留了具体的profile相关功能词汇，确保精准匹配
